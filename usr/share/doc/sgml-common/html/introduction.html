Chapter 1. Introduction

The SGML-common package is a collection of very basic stuff useful for processing SML documents. It is developped by many authors and released under GPL licence.

It currently contains the ISO character entities for SGML and a script named install-catalog. It's very unlikely that new elements will appear, because this package must remain minimal.
