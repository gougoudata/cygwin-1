stg-prev(1)
===========

NAME
----
stg-prev - Print the name of the previous patch

SYNOPSIS
--------
[verse]
'stg' prev 

DESCRIPTION
-----------

Print the name of the previous patch.

OPTIONS
-------
-b BRANCH::
--branch BRANCH::
        Use BRANCH instead of the default branch.

Part of the StGit suite - see linkman:stg[1]
