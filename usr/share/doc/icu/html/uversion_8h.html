#define U_NAMESPACE_BEGIN

This is used to begin a declaration of a public ICU C++ API.

When not compiling for C++, it does nothing. When compiling for C++, it begins an extern "C++" linkage block (to protect against cases in which an external client includes ICU header files inside an extern "C" linkage block). If the C++ compiler supports namespaces, it also begins a namespace block.
